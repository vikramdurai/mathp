# mathp
Math problem generation.

## API Calls and Parameters
- **/api/**:
    the API endpoint

### Parameters

|   Key    |    Sample Values   |    Type     |
|----------|--------------------|-------------|
| Grade    |    `5`,`3`,`6`     |    `int`    |
| Syllabus |   `NCERT`,`CBSE`   |   `string`  |
| Mode     |`Algebra`,`Geometry`|   `string`  |
| Pattern  |   `a² + b² = c²`   |   `string`  |
